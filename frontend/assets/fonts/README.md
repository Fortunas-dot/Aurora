# Fonts Directory

## Unbounded Regular

To use the Unbounded Regular font for headers, please download the font file and place it here:

1. Download `Unbounded-Regular.ttf` from Google Fonts:
   - Visit: https://fonts.google.com/specimen/Unbounded
   - Click "Download family"
   - Extract the ZIP file
   - Copy `Unbounded-Regular.ttf` to this directory: `frontend/assets/fonts/`

2. The font will be automatically loaded when the app starts.

**Note:** The font file should be named exactly `Unbounded-Regular.ttf` (case-sensitive).
